We recommend using the install-dependencies.sh script (included in the download) to install the dependencies and all plugin languages with one command.

But if you want to get just what you need, then run these commands from a terminal:

Java 7:

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer

nginx:

sudo add-apt-repository ppa:nginx/stable
sudo apt-get update
sudo apt-get install nginx
