Python

Our Python plugin requires a version of IPython >= 1.0. Install with:

sudo apt-get install ipython-notebook

Check your ipython version with ipython --version if it's below 1.0, run the following:

sudo apt-get install python-pip
sudo pip install --upgrade ipython[all] jinja2 tornado
R

Run this in your terminal:

sudo apt-get install r-base

Open R and run this:

install.packages('Rserve')
install.packages("ggplot2")
