Python
First install zmq3:
sudo add-apt-repository --yes ppa:chris-lea/zeromq
sudo apt-get update
sudo apt-get install -y libzmq3-dbg libzmq3-dev libzmq3

Then IPython proper:
sudo apt-get install -y python-pip python-dev
sudo pip install ipython jinja2 tornado
sudo pip install pyzmq

And some useful libraries:
sudo apt-get install -y python-matplotlib python-scipy python-pandas

If you already had IPython, check your version with ipython --version if it's below 1.0, run the following:

sudo apt-get install python-pip
sudo pip install --upgrade ipython[all] jinja2 tornado
