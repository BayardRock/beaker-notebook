Installation

Mac

Dependencies

Java 7

Install JDK 7

Python

Beaker requires Python to startup, and the Python plugin requires IPython >= 1.0. The easiest way to install both of these is is with Anaconda. The default install will give you everything you need.

Languages

Some languages require you to install separate packages:

Python

See above for a link to install IPython via Anaconda

R

Install R. Open R and run these commands

install.packages('Rserve',,'http://www.rforge.net/')
install.packages("ggplot2")
Julia

Install IPython using Anaconda as mentioned above. Then download and install Julia. Finally open Julia and run this:

Pkg.add("IJulia")

Linux

This is been tested on Ubuntu 12.04 and 13.10. We expect it will work on other versions of Ubuntu, and should be translatable to other distros.

Dependencies

Run these commands from a terminal:

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer # javac -v = 1.7.0_51
cd /usr/lib/jvm
sudo ln -s java-7-oracle default-java
sudo add-apt-repository ppa:nginx/stable
sudo apt-get update
sudo apt-get install nginx # nginx -v = 1.4.5
sudo apt-get install -y python

Languages

Some languages require you to install separate packages:

Python

Our Python plugin requires a version of IPython >= 1.0. Install with:

sudo apt-get install python-pip python-dev
sudo pip install ipython jinja2 tornado # ipython -v = 1.2.0
sudo pip install pyzmq --install-option="--zmq=bundled"
sudo apt-get install python-matplotlib python-scipy python-pandas
R

Run this:

sudo add-apt-repository ppa:marutter/rrutter
sudo apt-get update
sudo apt-get install r-base r-base-dev # R -v = 3.0.2
install.packages('Rserve',,'http://www.rforge.net/')
install.packages("ggplot2")

Using Beaker

After installation, just double click 'beaker.command' to get started. Beaker will automatically launch in your browser.

You'll probably want to start by checking out the tutorial notebook. Find it under the 'Help' menu at the top of the page.
