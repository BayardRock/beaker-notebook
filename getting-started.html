Using Beaker

The first time you run Beaker, control-click on it and select "open" from the menu, then click through the warning about being from an unknown developer. Subsequently you can just double click like any application.

It will run the server and open your web browser with a connection to it. When you exit the native application, the server quits too. If you need to debug a problem, you can find the logs with the Console app.

Using Beaker

After installation, just double click beaker.command.bat. It will open a terminal where the server runs, and a web browser with a connection to it. When you are done, kill the server by pressing Control-C in the terminal.

This is been tested on Ubuntu 12.04 LTS and 14.04 LTS. We expect it will work on other versions of Ubuntu, and should be translatable to other distros.
