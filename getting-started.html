This is been tested on Ubuntu 12.04 LTS and 14.04 LTS. We expect it will work on other versions of Ubuntu, and should be translatable to other distros.

Prerequisites

Run these commands from a terminal:

Java 7:

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java7-installer

nginx:

sudo add-apt-repository ppa:nginx/stable
sudo apt-get update
sudo apt-get install nginx

Download Beaker

You'll receive a download link when you confirm your email.

Languages

Some languages require a packages:

Python

Our Python plugin requires a version of IPython >= 1.0. Install with:

sudo apt-get install ipython-notebook

Check your ipython version with ipython --version if it's below 1.0, run the following:

sudo apt-get install python-pip
sudo pip install --upgrade ipython[all] jinja2 tornado
R

Run this in your terminal:

sudo apt-get install r-base

Open R and run this:

install.packages('Rserve',,'http://www.rforge.net/')
install.packages("ggplot2")
Julia

Requires IJulia

Using Beaker

After installation, just double click beaker.command (in Linux, you may need to call ./beaker.command from a terminal)

Start by checking out the tutorial notebook. Find it under the 'Help' menu at the top of the page.
