Installing and launching Beaker

Unsupported major.minor version

If you see an error like "java.lang.UnsupportedClassVersionError: : Unsupported major.minor version 51.0", it means Java 7 can't be found. Please install JDK 7

Can I change the heap size for Groovy?

Yes. To customize the heap size, launch Beaker like this:

./beaker.command --plugin-option Groovy:-Xmx8g

General

Can I customize my default notebook?

Yes, create a notebook you like and save it to ~/.beaker/default.bkr. This will be loaded whenever you click 'new notebook.'

Why are you asking for my email address?

The primary purpose of the beta is to get user feedback on what needs to be improved before our full launch. We may periodically ask you to complete short questionnaires about your experience. We may also send up notifications when significant new features have been added.

You can unsubscribe from our email list at any time and continue using Beaker.

How does Beaker relate to IPython Notebook

If you only work in Python, you can consider Beaker an alternative UI to IPython. Our Python plugin runs an IPython kernel on the backend, so all of your code will work the way you expect it to. We do have some UI features that we think are compelling, so check it out!

If you work with multiple languages, or one language that isn't Python, we think you'll find Beaker much more pleasant. Beaker's core was built to be language agnostic, so all languages are treated equally within Beaker.

Since several of our plugins (Python, Julia, Ruby) are based on IPython kernels, we've been able to spend our time working on Beaker's UI and our other plugins. Many thanks to IPython for their great kernel architecture!
